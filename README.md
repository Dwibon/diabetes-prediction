# 🩺 Diabetes Prediction System

## Overview
This project uses Machine Learning to predict whether a patient is **diabetic or non-diabetic** based on medical input features.

It is a **binary classification problem** aimed at assisting early detection of diabetes using patient health data.

---

## Features
- Data preprocessing and cleaning  
- Train-test data splitting  
- Feature scaling using StandardScaler  
- Logistic Regression model implementation  
- Predictive system for real-time input  

---

## Machine Learning Model
- **Algorithm Used:** Logistic Regression  
- **Type:** Supervised Learning (Classification)  

---

## 📊 Dataset
The dataset contains medical information of patients, including:

- Pregnancies  
- Glucose  
- Blood Pressure  
- Skin Thickness  
- Insulin  
- BMI  
- Diabetes Pedigree Function  
- Age  

**Target Variable:**
- `0` → Non-diabetic  
- `1` → Diabetic  

---

## 🛠️ Technologies Used
- Python  
- NumPy  
- Pandas  
- Scikit-learn  
- Google Colab  

---

## Workflow
1. Load dataset  
2. Perform data preprocessing  
3. Split data into training and testing sets  
4. Apply feature scaling  
5. Train Logistic Regression model  
6. Evaluate model performance  
7. Build predictive system  

---

## ▶️ How to Run
1. Open the `.ipynb` file in Google Colab  
2. Run all cells step-by-step  
3. Provide input data in the prediction section  
4. View prediction result  

---

## 📈 Results
- Model Accuracy: **77%**  
- The model performs reasonably well in predicting diabetes based on input features
